# E-commerce-website-using-MERN-Razorpay
Fully functional and modern e-commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) with secure payments, product management, and user authentication.

+ User Registration & Login (JWT + bcrypt).
+ Update Profile / Manage Address.
+ View Orders, Order Status & Payment Details.
+ Product Details Page.
+ Add and remove from Cart.
+ Razorpay Integration (works on test mode).
